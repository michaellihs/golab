## golab project hooks get

Get project hook

### Synopsis


Get a specific hook for a project.

```
golab project hooks get [flags]
```

### Options

```
  -h, --help          help for get
      --hook_id int   The ID of a project hook
  -i, --id string     (required) The ID or URL-encoded path of the project
```
